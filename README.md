# Quantitative-salivary-amylase-pipeline
Calculates Amylase activity of a saliva sample using a spectrophotometer
builds a standard curve, calculates reducing sugar concentration and amylase activity per temperature condition
Inputs required: glucose concentrations and absorbances for standard curve, OD readings per temperature, incubation time, enzyme volume
Assumptions: DNS method, linear relationship between concentration and absorbance, blank correction done before input
Limitations: assumes exactly the DNS colorimetric assay, standard curve must be linear to be valid, duplicate readings per temperature
